# Titan View / 巨头视野 - 产品需求文档 (PRD)

## 1. 产品概述 (Product Overview)

### 1.1 产品愿景
打造一个**基于“大佬思维模型”的AI情报分析系统**。解决用户在碎片化信息时代的“信息焦虑”和“决策瘫痪”问题。通过模拟李嘉诚、马斯克等顶尖企业家的思维模型，对海量资讯进行过滤、重构和深度解读，为用户提供高价值的晨间决策仪表盘。

### 1.2 核心价值
*   **去噪 (De-noise)**：过滤掉90%的无关噪音，只保留高信噪比信息。
*   **视角 (Perspective)**：提供不仅是“发生了什么”，而是“大佬怎么看这件事”。
*   **效率 (Efficiency)**：每日晨间快速获取中国政策、国际形势、投资金融、AI科技等全方位信息。

---

## 2. 用户画像 (User Persona)
*   **目标用户**：投资者、创业者、企业高管、关注宏观经济与前沿科技的终身学习者。
*   **痛点**：
    *   每天被碎片化新闻轰炸，抓不住重点。
    *   看不懂政策背后的深层含义。
    *   错过关键的行业红利或风险预警。
    *   忙碌导致无法及时跟进全球动态。

---

## 3. 功能需求 (Functional Requirements)

### 3.1 “大佬视角”思维模型 (The Titan Models)
系统需内置并持续训练以下思维模型作为信息过滤器：

#### A. 李嘉诚视角 (The Tycoon / 稳健资本家)
*   **关键词**：#避险 #现金流 #政策风向 #地缘政治 #周期套利 #公用事业
*   **关注源**：发改委/央行文件、国际贸易战动态、房地产税、汇率波动、社会消费情绪（小红书/微博消费降级趋势）。
*   **输出风格**：谨慎、务实、关注风险收益比、长周期视角。

#### B. 马斯克视角 (The Visionary / 第一性原理)
*   **关键词**：#效率 #硬科技 #AI #能源 #火星 #自由意志 #第一性原理
*   **关注源**：arXiv论文、GitHub趋势、硅谷创投、电池技术、航天进展、出生率数据、全球文化冲突。
*   **输出风格**：激进、极客、关注技术突破、人类命运视角。

### 3.2 核心功能模块

#### 3.2.1 每日晨间决策简报 (Daily Morning Brief)
*   **描述**：每天早上生成一份结构化的深度简报。
*   **内容结构**：
    *   **红绿灯信号**：
        *   🔴 **风险预警**（李嘉诚）：如“汇率异常波动”、“新规限制”。
        *   🟢 **机会信号**（马斯克）：如“AI成本降低”、“新材料突破”。
    *   **四大板块深度解读**：
        1.  **中国政策**：政策原文 vs 落地影响。
        2.  **国际形势**：地缘政治对资本流动的影响。
        3.  **投资金融**：二级市场、大宗商品、加密货币关键异动。
        4.  **AI与科技**：真正具有颠覆性的技术进展。

#### 3.2.2 跨平台情报聚合 (Cross-Platform Intelligence)
*   **数据源覆盖**：
    *   **官方/新闻**：新华社、路透、彭博、政府官网。
    *   **社交/舆论**：
        *   **小红书**：挖掘社会情绪、消费趋势（如“备婚贴减少”、“转行热帖”）。
        *   **微博**：热搜背后的社会心理分析。
    *   **科技圈**：Hacker News, X (Twitter) 科技圈讨论。

#### 3.2.3 大佬模拟对话 (Ask the Titans)
*   **描述**：针对具体新闻，用户可点击查看不同大佬的AI模拟点评。
*   **交互**：用户提问“如何看待某政策”，系统分别输出“李嘉诚视角”和“马斯克视角”的回答。

---

## 4. 非功能需求 (Non-functional Requirements)
*   **响应速度**：每日简报需在早晨 8:00 前生成完毕。
*   **数据准确性**：必须标注信息来源（Source），杜绝AI幻觉。
*   **可扩展性**：支持未来增加更多大佬视角（如巴菲特、乔布斯）。

---

## 5. 数据需求
*   **存储**：所有爬取的原始数据及AI处理后的分析数据需按日期归档存储。
*   **保留期**：建议保留至少一年的历史数据以便回溯分析。
